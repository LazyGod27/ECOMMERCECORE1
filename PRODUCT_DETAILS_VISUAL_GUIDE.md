### Desktop (1200px+)
- 2-column grid for details
- Full-size tables and images
- All sections visible

### Tablet (768px - 1199px)
- 1-column grid for details
- Scaled tables and images
- Condensed layout

### Mobile (< 768px)
- Single column
- Stacked items
- Thumb-friendly buttons
- Tables scroll horizontally
